**转 Excel 文件为 List:**

```autohotkey
ExcelToList(FileName, nSheet := 1, last_row := "", last_column := "")
```

### 参数 Parameters: 

- FileName - 文件名
- nSheet - 工作表名
- last_row - 截止行(数字)
- last_column - 截止列(数字)

### 返回 Returns: 
List
### 抛出异常 Throws: 
- _Ex.NoExistFile
### 示例 Example

- 测试文件 `test.xlsx` 放在根目录了

```autohotkey
;全参数示例
arr := ExcelToList("test.xlsx","Sheet2","3","1")
```
